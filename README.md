# gift-money-inquiry
礼金查询系统

## 所有命令

使用本框架建议使用`yarn`进行安装启动，安装`yarn`请执行 `npm install yarn -g`

``` 安装
yarn install
```

### 启动
```
yarn dev
```

### 打包
```
yarn build
```
